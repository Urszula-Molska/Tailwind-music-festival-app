## Run Locally

Clone the project

```bash
  git clone https://github.com/rszula-Molska/Tailwind/tailwind-starter-course
```

Go to the project directory

```bash
 cd tailwind-starter-course
```

Open starter repo in another VSC window

```bash
 code .
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm tw:build
```
